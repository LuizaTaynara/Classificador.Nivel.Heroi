# 🦸 Classificador de Nível de Herói

## **O Que deve ser utilizado**
- Variáveis
- Operadores
- Estruturas de decisão
- Laços de repetição

---

## Objetivo 

Criar uma variável para armazenar o nome e a quantidade de experiência (XP) de um herói e depois utilizar uma estrutura de decisão para apresentar o nível do herói, de acordo com a regra de classificação.

---

## Regra de classificação

| XP | Nível |
|----|--------|
| Até 1.000 | Ferro |
| 1.001 - 2.000 | Bronze |
| 2.001 - 5.000 | Prata |
| 5.001 - 7.000 | Ouro |
| 7.001 - 8.000 | Platina |
| 8.001 - 9.000 | Ascendente |
| 9.001 - 10.000 | Imortal |
| Acima de 10.000 | Radiante |

---

## Exemplo de saída
O Herói de nome **Luiza** está no nível **Radiante**
